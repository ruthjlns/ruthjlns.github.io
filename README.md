# Ruth Julien Sutanto — Portfolio

Personal portfolio website showcasing graphic design, product design, app & web, and animation work.

🔗 **Live site:** [ruthjlns.github.io](https://ruthjlns.github.io)

## About

I'm Ruth Julien Sutanto, a graphic designer from Surabaya, Indonesia. This portfolio highlights my work across pattern design, packaging, merchandise, app/web design, and animation — built with Adobe Illustrator, Affinity, After Effects, and Photoshop.

## Built With

- HTML5 & CSS3 (no frameworks)
- Vanilla JavaScript
- Custom local fonts (Ethika, SansSerifBookFlf, SansSerifBold)
- BEM naming convention for CSS
- Intersection Observer API for scroll animations

## Features

- Responsive navigation with dropdown menu
- Scroll-triggered reveal animations
- Interactive project grid with hover blur + text overlay
- GIF animations embedded in project detail pages
- Lazy-loaded images for performance
